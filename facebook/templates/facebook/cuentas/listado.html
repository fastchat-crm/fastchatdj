¿Cómo conectar una página?
  1. Necesitas ser administrador de la página de Facebook.
  2. Genera un Page Access Token de larga duración con los permisos pages_messaging, pages_show_list, pages_manage_metadata y pages_manage_engagement.

  4. Configura el webhook en tu App de Meta con la URL y el verify token de la tarjeta, suscribiendo los campos messages y feed.

¿Cómo obtengo las credenciales? Pasos detallados
1
Sé administrador de la página La cuenta que genera el token debe administrar la página de Facebook que vas a conectar.
2
Genera un token de usuario en Graph API Explorer Entra a developers.facebook.com/tools/explorer, elige tu App y pide un User Access Token con los permisos pages_messaging, pages_show_list, pages_manage_metadata y pages_manage_engagement.
3
Conviértelo en token de larga duración Intercámbialo con fb_exchange_token usando el App ID y App Secret; el Page Access Token que devuelve /me/accounts con ese token ya no expira.
4
Pega el token abajo y pulsa "Detectar automáticamente" El sistema consulta /me/accounts y llena solos el Page ID y el nombre de la página.
5
Configura el webhook en tu App de Meta En Webhooks → Page usa la URL {{ webhook_url }} con el verify token que genera la tarjeta y suscribe los campos messages y feed.
