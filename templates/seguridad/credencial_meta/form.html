¿De dónde saco cada dato?

Todos estos datos se obtienen en Meta for Developers (con la cuenta Facebook del administrador del negocio).

  1. 1
    Meta App ID

    Entra a developers.facebook.com/apps → elige tu App → arriba a la izquierda, debajo del nombre, aparece el ID de la aplicación (un número de ~16 dígitos).

  2. 2
    Meta App Secret

    Dentro de la App → menú izquierdo App settings → Basic → campo App secret → pulsa Show (te pedirá tu password de Facebook).

    Nunca lo pegues en Slack, email ni chats. Al guardarlo aquí se cifra con Fernet automáticamente.
  3. 3
    Business Manager ID opcional

    Entra a business.facebook.com/settings → arriba a la izquierda, debajo del nombre del negocio, aparece el Business ID.

  4. 4
    System User ID opcional

    En Business Settings → Users → System Users → crea uno tipo Admin (si no existe) → su ID aparece al lado del nombre.

  5. 5
    System User Token opcional

    En la misma pantalla del System User → Generate new token → selecciona la App del paso 1 → marca los scopes:

    • whatsapp_business_management
    • whatsapp_business_messaging
    • business_management
    • pages_messaging (si usas Messenger)
    • instagram_manage_messages (si usas IG)

    Elige Never expires y copia el token una sola vez (Meta no lo vuelve a mostrar).

  6. 6
    Embedded Signup Config ID opcional

    Es el ID de la configuración de Facebook Login for Business que se abre cuando el cliente conecta su WhatsApp con un solo clic (flujo Embedded Signup). Dentro de la App → Facebook Login for Business → Configurations:

    URL directa: developers.facebook.com/apps → tu App → menú izquierdo Facebook Login for Business → pestaña Configurations → copia el Configuration ID (~16 dígitos).

    Si no existe, créala con Create configuration:

    • Login type: Business login
    • Asset: WhatsApp Business Account + Phone Number
    • Permisos: whatsapp_business_management, whatsapp_business_messaging
    Si dejas este campo vacío y pulsas Auto-detectar desde Meta, el sistema lo busca solo. Si hay varias configs, te deja elegir.
Requisitos en Meta antes de conectar
  • Business Manager verificado (o en proceso).
  • App en modo Live (no Development) para producción.
  • Producto WhatsApp → Cloud API agregado a la App.
  • System User con permiso Admin sobre la WABA.
