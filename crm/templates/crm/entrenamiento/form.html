{# ══════ Modal: Guía para obtener tokens (Gemini / OpenAI / Claude) ══════ #}
Cómo obtener un token (API Key) para IA
Un token / API Key es una credencial secreta que identifica tu cuenta ante el proveedor de IA. Lo copias, lo pegas en “+ Api Key” y lo usas para chat, audio o imagen. La clave no se comparte con nadie y puedes rotarla en cualquier momento.
{# ═══ GEMINI ═══ #}
Google Gemini
Plan Free disponible

Cuenta con plan gratuito generoso. Necesitas una cuenta Google y (opcionalmente) un proyecto en Google Cloud para upgrade a pago.

  1. Abre Google AI Studio: aistudio.google.com/app/apikey
  2. Inicia sesión con tu cuenta Google.
  3. Acepta los términos si es tu primera vez.
  4. Clic en “Crear clave de API” / "Create API key".
  5. Selecciona un proyecto existente o crea uno nuevo (ej. fastchat-ia).
  6. Copia la clave generada (empieza con AIzaSy...). Guárdala — no la verás completa de nuevo.
  7. Vuelve aquí → + Api Key → Proveedor: GEMINI → pega la clave → Modelo sugerido: gemini-2.5-flash.
Upgrade a plan pago (para evitar quotas de free tier): aistudio.google.com/app/plan_information · Habilita billing en Google Cloud Console para el proyecto de esa key.
Límites Free Tier: ai.google.dev/gemini-api/docs/rate-limits — varían por modelo. Si ves error 429, el plan quedó sin cupo.
{# ═══ OPENAI ═══ #}
OpenAI (GPT)
Pago por uso

OpenAI cobra por tokens consumidos. Necesitas cargar saldo o tarjeta de crédito antes de generar la API Key.

  1. Crea una cuenta o inicia sesión en: platform.openai.com/signup
  2. Configura método de pago en Billing: platform.openai.com/account/billing · Agrega tarjeta y carga un mínimo (ej. $5).
  3. Ve a API Keys: platform.openai.com/api-keys
  4. Clic en “Create new secret key”.
  5. Nómbrala (ej. fastchat-prod), elige un proyecto y permisos All (o restringe por scope si prefieres).
  6. Copia la clave (empieza con sk-...). Sólo se muestra una vez.
  7. Vuelve aquí → + Api Key → Proveedor: OPEN IA → pega la clave → Modelo sugerido: gpt-4o-mini (económico) o gpt-4o (calidad).
Rate limits & cuotas: platform.openai.com/docs/guides/rate-limits
Tip: Si recibes insufficient_quota, recarga saldo en Billing. Los límites son por organización y por modelo.
{# ═══ CLAUDE ═══ #}
Anthropic (Claude)
Pago por uso

Claude cobra por tokens consumidos. Anthropic ofrece saldo promocional al registrarte.

  1. Crea una cuenta o inicia sesión en: console.anthropic.com
  2. Verifica tu email y número de teléfono (requerido por Anthropic).
  3. Revisa/carga saldo en Billing: console.anthropic.com/settings/billing
  4. Ve a API Keys: console.anthropic.com/settings/keys
  5. Clic en “Create Key”.
  6. Nómbrala (ej. fastchat-prod) y elige el workspace.
  7. Copia la clave (empieza con sk-ant-...). Sólo se muestra una vez.
  8. Vuelve aquí → + Api Key → Proveedor: CLAUDE → pega la clave → Modelo sugerido: claude-haiku-4-5-20251001 (rápido) o claude-sonnet-4-5 (balanceado).
Importante: Claude no ofrece embeddings. Si tu agente usa vectorstore (FAISS con archivos PDF/CSV), agrega también una API Key de Gemini u OpenAI para ese vectorstore — Claude se encarga del chat.
Rate limits: docs.anthropic.com/claude/reference/rate-limits
{# ═══ Consejos generales ═══ #}
Consejos al manejar tokens
  • No compartas la clave ni la subas a GitHub. FastChat la almacena cifrada.
  • Configura alertas de consumo por key (botón ) para que te avisen si pasa un umbral.
  • Si tu plan free se agota, FastChat detecta el 429 y desactiva la key automáticamente para no seguir intentando.
  • Pulsa “Probar” () después de registrarla para validar que funciona con el modelo configurado.
  • Puedes tener varias keys del mismo proveedor (distintos proyectos, saldos separados).
